# Auxi Login

Aplicativo mobile desenvolvido com React Native + Expo para simular um fluxo de identificação, senha, seleção de condomínio e dashboard de funcionalidades.

## Tecnologias

- React Native
- Expo
- React Navigation
- TypeScript

## Estrutura do projeto

- `src/screens` — telas da aplicação
- `src/components` — componentes reutilizáveis
- `src/services` — integração com serviços e autenticação
- `src/theme` — tokens e estilos globais
- `src/utils` — utilidades auxiliares

## Como executar

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Inicie o projeto:
   ```bash
   npm start
   ```

3. Abra o app no emulador ou no dispositivo físico utilizando o Expo.

## Scripts disponíveis

- `npm start` — inicia o Expo
- `npm run android` — abre o app no Android
- `npm run ios` — abre o app no iOS
- `npm run web` — inicia a versão web

## Observações

Este projeto é um exemplo de fluxo de navegação e interface para um aplicativo de acesso a condomínios, com telas de identificação, senha, listagem e dashboard.
